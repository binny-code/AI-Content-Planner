# AI-Content-Planner
This project is an AI-based content planner that helps users create content strategies automatically. It analyzes a business domain, finds relevant keywords, and generates structured blog outlines. The tool reduces manual research and speeds up content planning. (This tool helps you write authentic blogs )



# AI Content Planner using CrewAI

This project implements a multi-agent AI Content Planner using CrewAI and Groq-hosted LLaMA 3.

## Features
- Content strategy generation
- SEO keyword research
- Blog outline creation
- Multi-agent orchestration
- Streamlit UI

python -m venv crewai-env
